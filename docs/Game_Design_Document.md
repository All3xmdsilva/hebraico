# VISÃO GERAL

## O que é o Hebraico Quest?

Hebraico Quest é uma plataforma gamificada para ensino de Hebraico Bíblico.

O objetivo não é apenas ensinar um idioma.

O objetivo é conduzir o estudante desde o alfabeto até a leitura do Antigo Testamento em hebraico através de uma jornada de aprendizado.

---

# MISSÃO DO JOGADOR

O jogador inicia sem qualquer conhecimento.

Ao longo da aventura ele deverá:

✔ aprender o alfabeto

✔ dominar as vogais

✔ formar sílabas

✔ ler palavras

✔ entender gramática

✔ interpretar frases

✔ ler versículos

✔ ler livros completos da Bíblia Hebraica

---

# HISTÓRIA

O jogador chega à antiga Jerusalém como um aprendiz.

Um antigo mestre lhe entrega um pergaminho.

Nesse pergaminho existe um mapa do conhecimento perdido.

Cada região representa um nível do idioma.

Somente dominando cada etapa será possível compreender totalmente os escritos antigos.

# FILOSOFIA

O Hebraico Quest existe para mostrar que aprender Hebraico Bíblico pode ser prazeroso, organizado e acessível.

A gamificação nunca substituirá o conteúdo.

Ela servirá para aumentar a motivação e a constância dos estudos.

Todo recurso implementado deverá favorecer o aprendizado e respeitar o conteúdo linguístico.
