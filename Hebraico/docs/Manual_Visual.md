# Objetivo

Este documento define toda a identidade visual do Hebraico Quest.

Toda tela criada deverá seguir este padrão.

---

# Filosofia

O Hebraico Quest não é apenas um curso.

É uma aventura de aprendizado.

A interface deve transmitir:

• calma

• organização

• descoberta

• progresso

• recompensa
