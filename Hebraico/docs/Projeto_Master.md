# 1. VISÃO GERAL

## Nome do Projeto

Hebraico Quest

## Slogan

Aprenda Hebraico Bíblico como uma grande aventura.

## Descrição

O Hebraico Quest é uma plataforma gamificada para o ensino de Hebraico Bíblico, desenvolvida para funcionar em computadores, celulares e tablets.

Seu objetivo é unir ensino estruturado, jogos educativos, recursos multimídia e acompanhamento do progresso do aluno em uma única aplicação.

O projeto será desenvolvido inicialmente utilizando HTML, CSS e JavaScript e posteriormente evoluirá para uma versão online utilizando Node.js, Express e MongoDB, além de uma versão em Flutter para Android, iOS e Desktop.

# 15. FILOSOFIA DO PROJETO

O Hebraico Quest não busca apenas ensinar um idioma.

Seu propósito é proporcionar uma experiência completa de aprendizado do Hebraico Bíblico, combinando conhecimento linguístico, contexto bíblico, gamificação e tecnologia em uma plataforma acessível, organizada e agradável de utilizar.

Todo recurso implementado deverá contribuir para esse objetivo, mantendo equilíbrio entre qualidade pedagógica, usabilidade e evolução contínua do aluno.
